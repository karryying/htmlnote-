关于我们

优就业是中公教育IT培训品牌，致力于培养面向互联网领域的高端人才，以学员就业为目的，优质就业为宗旨。培训领域覆盖互联网营销师、UI交互设计师、Web前端工程师、Java工程师、PHP工程师、SEO优化师、SEM竞价师、社会化媒体运营师等。

职业规划

优就业职业规划测评系统，根据个人职业兴趣、能力及自我角色的认知，快速职场定位，并制定切实可行的计划。让学员都能按照自己的定位、兴趣和能力，找到适合自己的平台，以此实现自我的职场目标，创造价值。

就业服务

学员学完培训规定内课程，中公优就业将根据学员学习情况、学员自身意愿，提供就业服务，提升求职面试技巧，增加面试成功几率，帮助学员找到理想工作。更有名企招聘双选会， 企业登门招聘，学员与企业面对面交流。在就业求职的路上，优就业助你一臂之力。
